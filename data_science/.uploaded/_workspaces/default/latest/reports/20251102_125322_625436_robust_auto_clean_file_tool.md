## Dataset Tools — Quick Start

Use these functions in the next turn to work with your bound dataset:

- `list_data_files()` — list CSV files available
- `analyze_dataset()` — auto-EDA summary
- `describe()` — column-wise stats
- `shape()` — dataset dimensions
- `robust_auto_clean_file()` — file-based cleaner (ADK-safe)
- `impute_simple(strategy='mean'|'median'|'most_frequent')` — basic imputation
- `encode_data()` / `scale_data()` — prep features
- `train_classifier(target='YOUR_COLUMN')` — quick baseline classifier
- `train_regressor(target='YOUR_COLUMN')` — quick baseline regressor
- `export_executive_report()` — generate an executive PDF/MD report
